## Project Overview

This project demonstrates the implementation of fundamental Machine Learning concepts, including train/test splitting, overfitting, underfitting, the bias-variance tradeoff, Linear Regression, cross-validation, learning curves, and Polynomial Regression using Python and Scikit-learn.

---

## Dataset

- California Housing Dataset (Scikit-learn)

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Tasks Performed

- Loaded and explored the dataset.
- Performed train/test splitting.
- Implemented Linear Regression.
- Evaluated the model using MSE and R² Score.
- Compared different train/test split ratios.
- Performed 5-fold cross-validation.
- Plotted a learning curve.
- Implemented Polynomial Regression (Bonus Task).
